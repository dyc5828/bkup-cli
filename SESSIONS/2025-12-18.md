# 2025-12-18: Output & Restore Enhancements (v1.1.0)

Simplified output format and added smarter restore functionality.

## Changes

- Replaced verbose labels with git-style `+`/`-` markers
- Added auto-detection of backup files in restore mode
- Dry-run now shows `[dry-run]` header once at start

## Output Format

| Mode | Output |
|------|--------|
| Copy | `+ file.bkup` |
| Move | `+ file.bkup` then `- file` |
| Restore | `+ original` then `- original.bkup` |

## Smarter Restore

- `bkup -r file.txt` now auto-finds `file.txt.bkup`
- Falls back to timestamped backups if exact match not found
- Picks most recent when multiple timestamped backups exist
- Works with custom extensions (`-e`)
- Shows warning if no backup found
