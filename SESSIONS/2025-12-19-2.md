# 2025-12-19: Zsh Completions (v1.3.0)

Added zsh tab completions with Homebrew integration.

## Changes

- Created `completions/_bkup` for zsh tab completion
- Updated Homebrew formula to install completions automatically

## Completion Behavior

| Flag | Behavior |
|------|----------|
| `-d, --delete` | Mutually exclusive with `-r` |
| `-r, --restore` | Mutually exclusive with `-d` |
| `-e, --extension` | Expects extension argument |
| Other flags | Toggle options |
| Arguments | Files and directories |

## Homebrew Integration

- Formula installs to `$(brew --prefix)/share/zsh/site-functions/_bkup`
- Completions auto-load for users
